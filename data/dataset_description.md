# Dataset Description

Source:

Google Merchandise Sales Data (Kaggle)

Dataset URL:
https://www.kaggle.com/datasets/mexwell/google-merchandise-sales-data

Description:

This dataset contains a curated subset of anonymized Google Analytics ecommerce event data from the Google Merchandise Store.

The dataset includes:

* User activity events
* Product catalog information
* User lifetime value (LTV)
* Device information
* Country information
* Ecommerce conversion events

Files:

events1.csv

* user_id
* ga_session_id
* country
* device
* type
* item_id
* date

items.csv

* id
* name
* brand
* variant
* category
* price_in_usd

users.csv

* id
* ltv
* date

The original dataset was merged and transformed into a single analytical table used for Power BI reporting.
